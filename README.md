# AI Models 🚀

A comprehensive repository for learning, building, fine-tuning, evaluating, and deploying modern Artificial Intelligence systems.

## Overview

This repository serves as my end-to-end AI Engineering workspace where I study, implement, and deploy modern AI systems ranging from traditional Machine Learning models to Large Language Models (LLMs), Agentic AI systems, and Production AI Infrastructure.

The goal is to progress from intermediate AI knowledge to an industry-level AI Engineer capable of building and deploying production-ready AI applications.

## Environment Setup

### Clone Repository

```bash

```

### Create Environment

```bash
conda env create -f environment.yml
```

### Activate Environment

```bash
conda activate AI
```
